# passport-yandex

Пакеты для запуска

* npm install -S express
* npm install -S express-session
* npm install -S passport
* npm install -S passport-yandex
* npm install -S passport-google-oauth20

Старт 

* node index.js
* заходим на http://localhost:8081/
